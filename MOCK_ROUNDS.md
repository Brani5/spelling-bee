## How to Run a Mock Round

### Setup
- Parent = Pronouncer (judge)
- Otto = Contestant
- Optional: stuffed animal or sibling as "audience"

### Rules (simplified for 7-year-old)
1. Pronouncer says the word clearly
2. Pronouncer uses it in a sentence
3. Otto may ask:
   - **"Can you repeat the word?"** (unlimited)
   - **"Can you use it in a sentence?"** (if he missed it)
4. Otto spells the word OUT LOUD, letter by letter
5. Otto says the word again after spelling: "B-R-I-D-G-E, bridge."
6. Pronouncer says **"That is correct!"** or **"Nice try! The correct spelling is..."**

### Scoring
- ‚úÖ = Correct on first try (2 points)
- üîÑ = Correct on second try (1 point)  
- ‚ùå = Needed help (0 points, but NO elimination ‚Äî we keep going!)
